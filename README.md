
# Rostom Siervo - Smart Resume

This is a professional, interactive resume application built with React and the Google Gemini API.

## Deployment on Vercel

1. **Connect your Repository**: Push this code to a GitHub, GitLab, or Bitbucket repository.
2. **Import to Vercel**: Go to [Vercel](https://vercel.com) and click "New Project", then select your repository.
3. **Configure Environment Variables**:
   - In the "Environment Variables" section during setup (or in Project Settings later), add a new key.
   - **Key**: `API_KEY`
   - **Value**: Your Google Gemini API Key (get one at [aistudio.google.com](https://aistudio.google.com/)).
4. **Deploy**: Click "Deploy". Vercel will automatically detect the Vite configuration and build your site.

## Features
- **AI Career Assistant**: Ask specific questions about the resume.
- **Dark Mode UI**: Professional high-end aesthetic.
- **Print Optimized**: Export as a clean, industry-standard PDF.
- **Responsive**: Works on all screen sizes.
